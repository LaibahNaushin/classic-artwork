# Classic Artwork Fetcher

A modern and responsive web application that allows users to explore classic artworks from the Metropolitan Museum of Art Collection. Users can search artworks, discover random masterpieces, save favorites, copy artwork details, download information, and enjoy a clean user experience with dark mode support.

## Features

- Search artworks by keyword
- Display featured artwork on page load
- Explore random artworks
- Loading indicator during API requests
- User-friendly error handling
- Empty state messages
- Recent search history
- Save favorite artworks
- Copy artwork details
- Download artwork details as a text file
- Dark Mode
- Responsive design for all devices
- Input sanitization for security
- Console analytics simulation
